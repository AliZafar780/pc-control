## ✨ Features

### 🤖 AI-Powered Intelligence
- **Smart Summarization** — AI-generated summaries of long posts
- **Content Analysis** — AI-powered engagement scoring
- **Quick Insights** — Extract key points without reading everything
- **Multiple Providers** — OpenAI, Anthropic, and local models

### 🎯 Distraction-Free Design
- **Focus Mode** — Strip away likes, comments, distractions
- **Content Filtering** — Filter by keywords, topics, sentiment
- **Reading Time Estimates** — Know content length upfront
- **Minimal Interface** — Clean, terminal-first experience

### 📱 Multi-Platform Support
- **Twitter/X** — Read without engagement pollution
- **Reddit** — Browse with AI summarization
- **LinkedIn** — Professional content without distractions
- **Hacker News** — Tech news with smart ranking

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/AliZafar780/pc-control.git
cd pc-control

# Install dependencies
pip install -r requirements.txt

# Run the CLI
python main.py
```

## ⚙️ Configuration

Set up your API keys in `.env`:

```bash
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-...
TWITTER_BEARER_TOKEN=...
REDDIT_CLIENT_ID=...
REDDIT_CLIENT_SECRET=...
```
